<h1 align="center">📊 HR Analytics Dashboard </h1>

<p align="center">
Interactive HR Analytics Dashboard using Tableau
</p>

<hr>

<h2>📌 Overview</h2>

<p>
This project focuses on analyzing employee workforce data using Tableau to identify key HR metrics,
employee attrition patterns, job satisfaction trends, and workforce demographics.
The dashboard provides interactive visualizations that help HR teams make data-driven decisions.
</p>

<h2>⚙️ Project Workflow</h2>

<h3>1️⃣ Data Preparation</h3>

<ul>
  <li>Imported HR employee dataset into Tableau</li>
  <li>Validated data fields and attributes</li>
  <li>Prepared data for analysis and visualization</li>
</ul>

<h3>2️⃣ Data Analysis</h3>

<ul>
  <li>Analyzed employee attrition trends</li>
  <li>Examined department-wise employee turnover</li>
  <li>Studied gender distribution and workforce demographics</li>
  <li>Evaluated job satisfaction ratings across job roles</li>
</ul>

<h3>3️⃣ Dashboard Development</h3>

<ul>
  <li>Created KPI cards for key HR metrics</li>
  <li>Built interactive charts and visualizations</li>
  <li>Added education-level filtering functionality</li>
  <li>Integrated dashboard navigation and export options</li>
</ul>
